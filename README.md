## Install

Packages with
```sh
npm install
```
or
```sh
yarn
```

Expo with 
```sh
npm install -g expo
```
or
```sh
yarn global add expo
```

## Usage

```sh
npm run start 
```
or
```sh
yarn start
```

## Run tests

```sh
npm run test
```
or
```sh
yarn test
```
